ember-wired
==============================================================================

This is a simple wrapper library around WiredJS web components.

https://wiredjs.com/

Compatibility
------------------------------------------------------------------------------

* Ember.js v2.18 or above
* Ember CLI v2.13 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-wired
```


Usage
------------------------------------------------------------------------------

See https://wiredjs.com/
